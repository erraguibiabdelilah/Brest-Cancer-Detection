🚀 MODES DISPONIBLES
════════════════════

1. MODE RAPIDE (Par défaut - Recommandé)
   ----------------------------------------
   ⚡ Temps d'analyse: < 2 secondes
   ✅ Contenu: Prédéfini et optimisé
   ✅ Qualité: Professionnelle
   ✅ Fiabilité: 100%
   
   Configuration: USE_GEMINI_AI = False (dans config.py)

2. MODE GEMINI AI (Optionnel)
   ----------------------------
   🤖 Temps d'analyse: 5-10 secondes
   ✅ Contenu: Unique pour chaque analyse
   ✅ Qualité: Personnalisée
   ⚠️  Nécessite: Clé API Gemini
   
   Configuration: USE_GEMINI_AI = True (dans config.py)

📊 COMPARAISON
══════════════

Critère              | Mode Rapide | Mode Gemini AI
---------------------|-------------|----------------
Temps d'analyse      | < 2s        | 5-10s
Contenu unique       | Non         | Oui
Qualité médicale     | Excellente  | Excellente
Configuration requise| Aucune      | Clé API
Coût                 | Gratuit     | Gratuit*

* Quota gratuit Gemini: 60 requêtes/minute

🔧 CHANGER DE MODE
══════════════════

Pour activer le Mode Rapide (recommandé):
------------------------------------------
1. Ouvrez: backend/config.py
2. Changez: USE_GEMINI_AI = False
3. Redémarrez le serveur

Pour activer Gemini AI:
-----------------------
1. Configurez votre clé API dans backend/.env
2. Ouvrez: backend/config.py
3. Changez: USE_GEMINI_AI = True
4. Redémarrez le serveur
